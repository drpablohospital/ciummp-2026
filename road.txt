1. Technology Stack:
  Backend: Python 3.10+, Flask
  Database: SQLite (can be replaced with PostgreSQL for production)
  Payments: Stripe API (Checkout Sessions)
  QR Code: qrcode + Pillow
  PDF Generation: reportlab
  Frontend: HTML5, CSS (Tailwind CSS CDN), minimal JavaScript
  Deployment: Render (web service with Gunicorn)
